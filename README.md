# metrodex-web

Metrodex web is a website that, as its name indicates, allows user to collect and record the subway stations they have visited. 

it contains a map of the city currently visited, along with a pannel to record the progress of the user.
The website makes the distinction between passed through and visited for each station through color system (blue for passed and green for visited, red for neither).

Future extensions for the website are in discussion, for the cities of Lausanne and London.
